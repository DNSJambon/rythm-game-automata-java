## 2) Structuration du code (M, V, C, B)
Le dossier game/info3/game contient  deux principaux dossier Controller  et  Model. En Controller ,sur le dossier Actions ,  on trouvera toutes les classes aux actions utilisé  dans  le fichier automates.gal , et de  même  pour  les conditions  ,  on trouverra toutes conditions nécessaire  pour nos automates .
En Model ,il existe  le fichier Automates.gal  qui décrit  le  comportement  de chaque  Entité  qui existe  dans  notre  moteur de  jeux  .D'autres  part  , surcontient les classes des entités (une classe par entité) dans le sous-dossier Entities. 
Il contient aussi la view dans le fichier Grille.java qui effectue l’affichage graphique des actions via les avatars associées aux entités.
